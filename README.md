# @royrai/ui-kit

Reusable React/TypeScript UI component library built on top of MUI (Material-UI) with bilingual (Hebrew/English) support.

## Installation

```bash
yarn add @royrai/ui-kit
```

### Peer Dependencies

This library requires the following peer dependencies in your project:

```bash
yarn add react react-dom @mui/material @mui/icons-material @mui/x-data-grid @mui/x-date-pickers @emotion/react @emotion/styled
```

## Usage

```tsx
import { RrBox, RrDataGrid } from "@royrai/ui-kit";
```

## Components

### Controls
- `RrBox` - Flexible box/container wrapping MUI Box
- `RrDataGrid` - Data grid wrapper with RTL support
- `RrSocialIcons` - Social media SVG icons (WhatsApp, Instagram, Facebook, LinkedIn, Email)
